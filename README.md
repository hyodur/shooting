# 시간 비행 드릴

React + Vite + TypeScript + Tailwind CSS로 만든 초등학교 3학년용 "시간의 덧셈과 뺄셈" 세로셈 드릴 게임입니다.

## 실행 방법

```bash
npm install
npm run dev
```

배포용 빌드는 다음 명령으로 확인합니다.

```bash
npm run build
```

로컬에서 배포 결과를 미리 확인하려면 다음 명령을 사용합니다.

```bash
npm run preview
```

## GitHub Pages 배포와 공유

이 앱은 Vite 정적 앱이라 GitHub Pages로 배포할 수 있습니다. 현재 프로젝트에는 GitHub Actions 기반 Pages 배포 설정이 포함되어 있습니다.

배포 순서:

1. GitHub 저장소의 `Settings > Pages`로 이동합니다.
2. `Build and deployment` 소스를 `GitHub Actions`로 선택합니다.
3. `main` 브랜치에 push하면 `.github/workflows/deploy-pages.yml`이 자동으로 빌드와 배포를 실행합니다.

배포 후 생성된 GitHub Pages URL을 다른 선생님들에게 공유하면 브라우저에서 바로 사용할 수 있습니다. 현재 랭킹은 각 기기의 브라우저 저장소에 저장되므로, 여러 학급이나 여러 기기에서 같은 랭킹을 공유하려면 서버 저장소 연결이 필요합니다.

## 앱 설명

- 학생은 "몇 시간 몇 분" 형태의 덧셈과 뺄셈 문제를 세로셈으로 풉니다.
- 정답의 시간 칸과 분 칸을 각각 입력합니다.
- 화면 숫자패드와 키보드 입력을 모두 사용할 수 있습니다.
- 정답이면 미사일이 발사되고 적 비행기를 격추합니다.
- 우주 배경, 적 등장 애니메이션, 효과음, 간단한 BGM 루프를 제공합니다.
- 점수는 정답, 풀이 속도, 콤보에 따라 올라가고 오답 시 감점됩니다.
- 결과 화면에서 난이도별 랭킹을 등록할 수 있습니다.
- 같은 문제를 2번 틀리면 하트가 1개 줄어듭니다.
- 하트가 0개가 되면 게임 오버입니다.
- 적 비행기 10대를 격추하면 스테이지 클리어입니다.

## 점수와 랭킹

- 정답을 맞히면 기본 점수를 얻습니다.
- 빨리 풀수록 시간 보너스가 커집니다.
- 연속 정답을 이어가면 콤보 보너스가 붙습니다.
- 오답은 바로 감점됩니다.
- 같은 문제에서 틀린 뒤 맞히면 정답 점수도 일부 줄어 정확하게 푸는 편이 유리합니다.
- 랭킹은 서버가 아니라 현재 기기의 브라우저 저장소에 쉬움, 보통, 어려움 난이도별로 저장됩니다.
- 여러 기기에서 공유되는 서버 랭킹을 쓰려면 Supabase, Firebase 같은 별도 저장소 연결이 필요합니다.

## 수학 규칙

시간 계산은 내부적으로 전체 분으로 바꾸어 처리합니다.

- `3시간 45분`은 `225분`입니다.
- 덧셈에서 분의 합이 60분 이상이면 1시간으로 받아올림합니다.
- 뺄셈에서 분끼리 뺄 수 없으면 1시간을 60분으로 받아내림합니다.
- 정답은 항상 "몇 시간 몇 분" 형태로 보여줍니다.
- 분 입력값은 0 이상 59 이하만 정답 판정에 사용할 수 있습니다.

## 난이도

- 쉬움: 받아올림이나 받아내림이 없는 시간의 덧셈과 뺄셈
- 보통: 받아올림 또는 받아내림이 한 번 필요한 문제
- 어려움: 60분과 1시간의 관계를 더 신중하게 생각해야 하는 문제

## 수업 활용 방법

- 전자칠판에서 한 문제씩 함께 풀며 "60분 = 1시간" 규칙을 확인할 수 있습니다.
- 태블릿에서는 학생이 혼자 문제를 풀고, 오답 힌트를 보고 다시 시도할 수 있습니다.
- 첫 오답에는 힌트만 제공하고, 두 번째 오답 후에는 정답과 간단한 풀이 과정을 보여줍니다.
- 결과 화면의 "어려웠던 유형 안내"를 보고 받아올림 덧셈 또는 받아내림 뺄셈을 추가 연습할 수 있습니다.

## 사운드 교체

현재 효과음과 BGM은 외부 파일 없이 Web Audio API로 합성합니다. 직접 만든 배경음악 파일이 있다면 `public/audio/`에 mp3 또는 wav 파일을 넣고, 앱에서 `<audio loop>` 방식으로 연결해 교체할 수 있습니다.

## 주요 파일

- `src/logic/problemGenerator.ts`: 난이도별 시간 덧셈, 뺄셈 문제 생성
- `src/logic/answerChecker.ts`: 최종 정답 판정과 힌트, 풀이 메시지
- `src/logic/timeUtils.ts`: 전체 분 변환, 시간 계산 유틸리티
- `src/logic/gameRules.ts`: 하트, 오답, 스테이지 클리어 규칙
- `src/components/TimeVerticalProblem.tsx`: 시간과 분이 분리된 세로셈 입력 UI
