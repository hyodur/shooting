# 시간비행대: 60분 작전

React + Vite + TypeScript + Tailwind CSS로 만든 초등학교 3학년용 시간 덧셈·뺄셈 세로셈 슈팅 드릴입니다.

## 실행

```bash
npm install
npm run dev
npm run build
```

`main` 브랜치에 변경 사항을 올리면 GitHub Actions가 GitHub Pages를 자동 배포합니다.

## 게임 규칙

- 문제를 맞히면 미사일을 발사해 적 비행기를 격추합니다.
- 같은 문제를 두 번 틀리면 하트가 1개 줄어듭니다.
- 하트가 0개가 되면 게임 오버, 적 10대를 격추하면 스테이지 클리어입니다.
- 빠른 풀이, 연속 정답, 정확한 풀이칸 입력에 보너스 점수를 줍니다.
- 변환 풀이칸을 틀리면 감점되므로 아무 숫자나 적는 것보다 비워 두는 편이 낫습니다.

## 난이도

- 쉬움: `시간·분` 또는 `분·초` 두 단위를 사용하며 받아올림·받아내림이 없습니다.
- 보통: `시간·분` 또는 `분·초` 두 단위를 사용하며 받아올림·받아내림이 정확히 한 번 있습니다.
- 어려움: `시간·분·초` 세 단위를 모두 사용하며 받아올림·받아내림이 정확히 한 번 있습니다.

## 세로셈 풀이칸

- 덧셈의 받아올림은 바로 왼쪽 단위 위 작은 칸에 `1`을 적습니다.
- 뺄셈에서 받아내리면 작은 칸 두 개를 사용합니다.
- 빌려준 단위 위에는 원래 수보다 `1 작은 수`를 적습니다.
- 빌린 단위 위에는 `60`을 적습니다.
- 예: `15분 25초 - 10분 30초`는 분 위에 `14`, 초 위에 `60`을 적습니다.
- 풀이칸을 정확히 쓰면 보통 `+350점`, 어려움 `+650점`, 틀리면 `-60점`입니다.

## 수학 처리

시간 계산은 내부적으로 전체 초로 변환하여 판정합니다.

- `3시간 45분 20초 = 13,520초`
- `60초 = 1분`, `60분 = 1시간`
- 분과 초의 최종 답은 각각 `0~59`만 허용합니다.

## 수업 활용

전자칠판에서는 한 문제씩 함께 풀며 변환 규칙을 확인하고, 태블릿이나 스마트폰에서는 가로 화면 사용을 권장합니다. 첫 오답에는 힌트가 나오고 두 번째 오답 뒤에는 정답과 간단한 풀이가 표시됩니다. 랭킹에는 실명 대신 번호나 별명을 사용하도록 안내해 주세요.
