# 시간비행대: 60분 작전

React + Vite + TypeScript + Tailwind CSS로 만든 초등학교 3학년용 "시간·분·초의 덧셈과 뺄셈" 세로셈 드릴 게임입니다.

## 실행 방법

```bash
npm install
npm run dev
```

배포용 빌드는 `npm run build`로 확인합니다.

## GitHub Pages 배포와 공유

이 앱은 Vite 정적 앱이라 GitHub Pages로 배포할 수 있습니다. `main` 브랜치에 변경 사항을 올리면 GitHub Actions가 자동으로 빌드하고 배포합니다.

## 앱 설명

- 학생은 "몇 시간 몇 분 몇 초" 형태의 덧셈과 뺄셈 문제를 세로셈으로 풉니다.
- 정답의 시간, 분, 초 칸을 각각 입력합니다.
- 화면 숫자패드와 키보드 입력을 모두 사용할 수 있습니다.
- 정답이면 미사일이 발사되고 적 비행기를 격추합니다.
- 점수는 정답, 풀이 속도, 콤보, 풀이칸 정확도에 따라 달라집니다.
- 같은 문제를 2번 틀리면 하트가 1개 줄어듭니다.
- 하트가 0개가 되면 게임 오버이며, 적 비행기 10대를 격추하면 스테이지 클리어입니다.
- 결과 화면에서 난이도별 전국 랭킹을 등록할 수 있습니다.

## 점수와 풀이칸

- 빨리 풀수록 속도 보너스가 커집니다.
- 연속 정답을 이어가면 콤보 보너스가 붙습니다.
- 오답은 바로 감점됩니다.
- 변환이 필요한 문제에서 풀이칸을 정확하게 쓰면 보통 `+350점`, 어려움 `+650점`을 받습니다.
- 풀이칸에 잘못 적으면 `-60점`, 비워 두면 추가 점수는 없습니다.

## 수학 규칙

시간 계산은 내부적으로 전체 초로 바꾸어 처리합니다.

- `3시간 45분 20초`는 `13,520초`입니다.
- 덧셈에서 초의 합이 60초 이상이면 1분으로, 분의 합이 60분 이상이면 1시간으로 받아올림합니다.
- 뺄셈에서 초끼리 뺄 수 없으면 1분을 60초로, 분끼리 뺄 수 없으면 1시간을 60분으로 받아내림합니다.
- 정답은 항상 "몇 시간 몇 분 몇 초" 형태로 보여줍니다.
- 분과 초 입력값은 각각 0 이상 59 이하만 정답 판정에 사용할 수 있습니다.

## 난이도

- 쉬움: 시간·분·초에서 받아올림이나 받아내림이 전혀 없는 문제
- 보통: 초→분 또는 분→시간 중 한 곳에서 정확히 한 번 변환하는 문제
- 어려움: 초→분과 분→시간 두 곳에서 정확히 두 번 변환하는 문제

## 수업 활용 방법

- 전자칠판에서 한 문제씩 함께 풀며 `60초 = 1분`, `60분 = 1시간` 규칙을 확인할 수 있습니다.
- 태블릿과 스마트폰에서는 가로 화면 사용을 권장합니다.
- 첫 오답에는 힌트만 제공하고, 두 번째 오답 후에는 정답과 간단한 풀이 과정을 보여줍니다.
- 랭킹에는 실명 대신 번호나 별명을 사용하도록 안내해 주세요.

## 사운드 교체

현재 효과음과 BGM은 외부 파일 없이 Web Audio API로 합성합니다. 직접 만든 배경음악은 `public/audio/`에 넣어 교체할 수 있습니다.
